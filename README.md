# ELK_Services_Script

### Change directory to ELK_Services_Script/elk_service:
```
$ cd ELK_Services_Script/elk_service
```

### 先更改執行檔的權限設定

```
$ chmod u+x s*
```

### Running ELK  :
```
$ ./start_elk.sh
```

### Checking  that ELK is running

```
$ ./status_elk.sh
```

### Stopping ELK:
```
$ ./stop_elk.sh
```
